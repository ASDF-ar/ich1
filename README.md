技术栈：
后端：SpringBoot3 + JWT + Druid + MyBatis

前端：Vue3 + ElementPlus + Axios

数据库： MySQL

版本要求：
jdk: 17或21

mysql: 8.0及以上

node.js:18以上

maven: 3.7以上

1. 管理员模块
登录功能

•数据统计展示
•数据统计柱状图
•分类统计饼图
后台首页

•管理员账号密码登录系统
非遗项目管理

•实时查看非遗项目信息列表
•支持按项目名称、类别、传承状态等条件查询
•支持项目详情编辑，包括项目名称、简介、详细介绍、图片、所在地、类别、传承状态
新闻资讯管理

•添加、编辑和删除新闻资讯
•管理新闻的标题、摘要、内容、封面、发布单位、类型等信息
•支持设置焦点新闻
活动预告管理

•添加、编辑和删除活动预告
•管理活动的名称、日期、时间、地点、主办单位、联系方式、活动详情等信息
项目申报类别管理

•添加、编辑和删除项目申报类别
•管理项目申报类别的名称等基本信息
项目申报管理

•添加、编辑和删除项目申报
•管理项目申报的名称、类别、简介、详细描述、传承谱系、传承人信息、联系方式等
传承人管理

•添加、编辑和删除传承人信息
•管理传承人基本信息（姓名、技艺专长、位置、照片等）
•记录传承人的传承贡献信息
关于我们管理

•基于数据位置在页面中实时编辑信息
•编辑关于我们信
•管理关于我们的内容，包括简介、使命、愿景、价值观、联系方式等
留言反馈管理

•添加、编辑和删除用户留言反馈
•管理留言的姓名、邮箱、留言内容等信息
轮播图管理

•管理用户前台的轮播图图片、标题、描述、排序
•添加、编辑和删除轮播图信息
2. 用户端模块
登录/注册

•账号注册和登录
个人中心

•查看和修改个人资料
•重置登录密码
前台首页

•查看轮播图展示的非遗项目和活动
•浏览推荐的非遗项目，包括项目图片、简介、位置和分类
•点击进入非遗项目详情页
非遗项目

•按分类筛选浏览非遗项
•查看所有非遗项目的列表展示，包括图片、标题、简介、位置和分类信
•点击项目查看详细信息
新闻资讯

•浏览焦点新闻，包含置顶新闻的特殊展示
•查看最新新闻列表
•点击新闻查看详细内容
活动预告

•查看活动详细信息，包括：活动标题和地点活动时间、地点、主办方和联系方式
•点击查看活动详细内容
传承人

•查看传承人列表，包括姓名、技艺专长和所在
•点击传承人查看详细信息
项目申报

•查看项目申报须知和要求
•在线填写申报信息
关于我们

•查看系统整体介绍
•了解系统的使命、愿景和价值观
•获取联系方式，包括电话、地址和邮箱
•提交留言反馈
